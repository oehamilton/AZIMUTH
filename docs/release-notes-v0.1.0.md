# AZIMUTH v0.1.0

First release. Windows 11 only.

## What's included

- **AZIMUTH Setup 0.1.0.exe** — Installer (NSIS). Recommended for most users.
- **AZIMUTH 0.1.0.exe** — Portable executable. No install; run from any folder.

## Features

- Home-centric world map (flat azimuthal) with bundled offline map
- Multiple saved home and target locations
- Set target by coordinates, saved list, or map click
- Short path and long path (great-circle) with bearing from North and distance
- Optional magnetic declination per home (compass bearing)
- Distance units: km, nm, miles
- Help → About: Project8X, Inc., support@project8x.com

## Requirements

- Windows 11
- No admin required for portable; installer can install for current user
